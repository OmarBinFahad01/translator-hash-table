# Multilingual Translator (Hash Table Implementation)

## Overview
This project is a multilingual dictionary implemented in C++ using hash tables. It supports efficient storage and retrieval of word translations across different languages.

## Features
- Custom hash table implementation (no STL hash maps)
- Multiple hash functions (Polynomial, Cyclic Shift, Division)
- Collision handling using probing
- Case-insensitive search
- Import/export dictionary files
- Add, delete, and search translations

## Commands
- find <word>
- import <file>
- add <word:meaning:language>
- delWord <word>
- delTranslation <word:language>
- delMeaning <word:meaning:language>
- export <language:filename>
